# Meet Shah — Portfolio

Live: **https://meet1906.github.io/**

A React + Vite + Tailwind single-page portfolio. The app lives in [`my-portfolio/`](./my-portfolio) and auto-deploys to GitHub Pages on every push to `master` via [`.github/workflows/deploy.yml`](./.github/workflows/deploy.yml).

## Local development

```bash
cd my-portfolio
npm install
npm run dev      # start dev server
npm run build    # production build → dist/
```
